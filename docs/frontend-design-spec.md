# 前端设计规范 — "数据报纸" 编辑风格

> 最后更新：2026-05-27
> 状态：已实现

---

## 设计概念

像阅读一份精心排版的数据报纸。品牌核心是"数据驱动，说真话"——用报纸编辑的严肃感+数据可视化的专业感。

**记忆点**：聊天界面排版像报纸专栏——AI 回复区有"专栏文章"的排版感，数据来源像"参考文献"脚注。

---

## 色彩系统

| 变量 | 色值 | 用途 |
|------|------|------|
| `--ink` | `#1a1a2e` | 深墨色 — 主文字/标题 |
| `--ink-light` | `#4a4a6a` | 浅墨色 — 次要文字 |
| `--gold` | `#c9a84c` | 暗金色 — 强调/高亮 |
| `--gold-light` | `#f0e6c8` | 浅金色 — hover/背景 |
| `--paper` | `#faf8f4` | 纸张米白 — 页面背景 |
| `--paper-dark` | `#f0ede6` | 深纸色 — 卡片背景 |
| `--red` | `#c23c2e` | 朱红 — 标签/警示 |
| `--rule` | `#e0ddd6` | 分割线 |

---

## 字体系统

| 用途 | 字体 | 来源 |
|------|------|------|
| 标题 | Noto Serif SC | Google Fonts — 衬线，报纸头条感 |
| 正文 | DM Sans | Google Fonts — 无衬线，清晰可读 |

## 页面设计

### 1. 首页（Portal）

**Masthead 报头区域**：
- 大标题用 Noto Serif SC 衬线字体
- 下方加日期线和卷号（如"第2026期 · 高考特刊"）
- 品牌标识："数据驱动 · 说真话 · 不画饼"

**三个场景卡片**：
- 排版像报纸分类广告
- 每个卡片有独特的 icon 和标签（"最热门"、"数据驱动"、"说真话"）
- 双线边框，hover 时有阴影提升效果

**底部数据快报横条**：
- 用报纸式的数据快报展示关键数据
- 格式：图标 + 数字 + 说明（如"3,000+ 院校真实数据"）

**装饰元素**：
- 报纸分割线（双线、单线、虚线）
- 卷号和日期线
- 报纸式页码

### 2. 表单页面（SoulQuestionForm）

- 排版像报纸问卷/读者调查表
- 进度条改为报纸版面标记（"第一版"、"第二版"等）
- 每个问题用报纸栏目的排版方式
- 输入框下方加"编者注"提示文字
- 步骤指示器用圆形编号 + 标签

### 3. 聊天界面（ChatInterface）

**布局**：
- 左侧聊天区占 2/3
- 右侧数据来源占 1/3
- 底部固定输入区

**消息排版**：
- 用户消息：右对齐，金色背景，像"读者来信"
- AI 回复：左对齐，纸张背景，衬线标题，像"专栏文章"
- 每条 AI 消息开头加装饰性的引号图标

**打字机效果**：
- 流式输出时逐字显示
- 带光标闪烁动画

**输入区**：
- 底部固定，像报纸的"读者来信"投稿区
- 标签："读者来信 · 投稿"

### 4. 数据来源面板（SourcePanel）

- 工具名称用等宽字体（Space Mono）显示
- 状态标签改为报纸式的红色印章风格
- 展开详情时显示结构化数据（表格形式），而非 JSON 原始数据
- 编号格式：[1]、[2]、[3]...

### 5. 反馈组件（FeedbackRating）

- 星级评分改为报纸式的"评分"印章风格
- 提交后显示"感谢来信"的报纸式确认

## 全局视觉元素

### 纸张纹理
CSS 实现的微妙纸张质感背景，使用渐变和噪点叠加。

### 报纸分割线
- 双线：`border-top: 3px double var(--rule)`
- 单线：`border-top: 1px solid var(--rule)`
- 虚线：`border-top: 1px dashed var(--rule)`

### 墨色渐变
标题和强调元素使用墨色渐变效果。

### 印章风格
红色背景 + 白色文字，用于状态标签和标记。

### 响应式
- 桌面端：报纸多栏布局
- 移动端：单栏布局，保持报纸感

## 技术实现

### 依赖
- React 18 + Vite + Tailwind CSS 3
- Google Fonts（Noto Serif SC、DM Sans、Space Mono）

### 文件结构
```
frontend/
├── index.html                    # Google Fonts 加载
├── tailwind.config.js            # 自定义色彩/字体
├── src/
│   ├── index.css                 # 全局样式
│   ├── App.tsx                   # 首页 Masthead
│   ├── components/
│   │   ├── ChatInterface.tsx     # 聊天界面
│   │   ├── MessageBubble.tsx     # 消息气泡
│   │   ├── SourcePanel.tsx       # 数据来源面板
│   │   ├── SoulQuestionForm.tsx  # 表单页面
│   │   └── FeedbackRating.tsx    # 反馈组件
```

### 设计原则
1. **只改视觉层，不改业务逻辑**
2. **保持现有功能完整**
3. **响应式适配**
4. **性能优先（CSS 动画，避免 JS 动画）**
